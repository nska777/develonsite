# Video People Counter

Одностраничный лендинг системы подсчёта посетителей с калькулятором лицензий по 15 500 ₽ за камеру и безопасным тестовым платёжным сценарием.

## Возможности

- адаптивный лендинг;
- макеты панели видеоаналитики;
- выбор от 1 до 20 камер;
- автоматический расчёт стоимости;
- тестовая форма оплаты без списания средств;
- тестовый итоговый документ операции;
- готово для статического размещения на Vercel.

## Развёртывание на Vercel

Импортируйте репозиторий как новый проект. Framework Preset: Other. Build Command оставьте пустым, Output Directory: `.`.
